## Installation

Add the Ourguide widget to your site with a single script tag:

<CodeGroup>

```html Script Tag
<script
  src="https://api.ourguide.com/static/ourguide-widget.js"
  data-api-url="https://api.ourguide.com"
  data-product-id="YOUR_PRODUCT_ID"
></script>
```

```jsx React SDK
import { OurguideWidget } from "@ourguide/react";

export default function App() {
  return (
    <OurguideWidget
      apiUrl="https://api.ourguide.com"
      productId="YOUR_PRODUCT_ID"
    />
  );
}
```

</CodeGroup>

## Core Features

| Feature                 | What it does                                                          |
| ----------------------- | --------------------------------------------------------------------- |
| **Knowledge Base**      | Agent learns from your docs, website, or Notion                       |
| **API Integration**     | Agent calls your API to take actions (check orders, cancel subscriptions) |
| **Identity Verification** | Agent knows who the user is and can act on their behalf             |
| **Workflows**           | Guided multi-step flows for complex tasks                             |

## How it Works

  <Step title="Embed">
