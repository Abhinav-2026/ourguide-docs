## Images

### Using Markdown

```md
![Alt text](/path/to/image.png)
```

### Using the Image Component

```html
<img src="/path/to/image.png" alt="Description" />
```

### Light and Dark Mode

Use different images for light and dark mode:

```html
<img className="block dark:hidden" src="/images/hero-light.svg" alt="Light mode" />
<img className="hidden dark:block" src="/images/hero-dark.svg" alt="Dark mode" />
```

## Supported Formats

- **PNG** — Best for screenshots and graphics with transparency
- **SVG** — Best for logos and icons (scales perfectly)
- **JPG** — Best for photographs
- **GIF** — Best for simple animations
- **WebP** — Modern format with great compression

<Tip>
  Store images in the `/images` directory at the root of your docs for best organization.
</Tip>
