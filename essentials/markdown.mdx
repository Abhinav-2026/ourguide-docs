## Titles

Best used for section headers.

```md
## Titles
```

### Subtitles

Best used for subsection headers.

```md
### Subtitles
```

<Tip>
  Each **page** should only have a single `#` heading, which is automatically set by the `title` in the frontmatter.
</Tip>

## Text Formatting

We support most markdown formatting. Simply add `**`, `## `, or `>` characters before your text to format it.

| Style         | How to write it   | Result          |
| ------------- | ----------------- | --------------- |
| Bold          | `**bold**`        | **bold**        |
| Italic        | `_italic_`        | _italic_        |
| Strikethrough | `~~strikethrough~~` | ~~strikethrough~~ |
| Code          | `` `code` ``      | `code`          |
| Link          | `[link](url)`     | [link](url)     |

## Blockquotes

> This is a blockquote. Best used for tips, notes, and callouts.

```md
> This is a blockquote
```
