## Sidebar Navigation

The sidebar is configured in the `navigation` field of `docs.json`:

```json
{
  "navigation": [
    {
      "group": "Get Started",
      "pages": ["introduction", "quickstart"]
    },
    {
      "group": "Essentials",
      "pages": [
        "essentials/markdown",
        "essentials/code"
      ]
    }
  ]
}
```

### Groups

Groups organize pages into collapsible sections. Each group has a `group` name and a `pages` array.

### Pages

Pages are referenced by their file path relative to the docs root, without the `.mdx` extension.

## Top Navigation

### Links

```json
{
  "topbarLinks": [
    {
      "name": "Support",
      "url": "mailto:support@yourcompany.com"
    }
  ]
}
```

### CTA Button

```json
{
  "topbarCtaButton": {
    "name": "Dashboard",
    "url": "https://dashboard.yourcompany.com"
  }
}
```

## Anchors

Anchors appear as tabs below the top navigation:

```json
{
  "anchors": [
    {
      "name": "API Reference",
      "icon": "rectangle-terminal",
      "url": "api-reference"
    }
  ]
}
```
