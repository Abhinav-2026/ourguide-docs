## 1. Copy your script tag

Go to [app.ourguide.com/deploy](https://app.ourguide.com/deploy) and copy your script tag.

## 2. Add to your site

<CodeGroup>

```html HTML
<script
  src="https://api.ourguide.com/static/ourguide-widget.js"
  data-api-url="https://api.ourguide.com"
  data-product-id="YOUR_PRODUCT_ID"
></script>
```

```jsx React / Next.js
import { OurguideWidget } from "@ourguide/react";

export default function App() {
  return (
    <OurguideWidget
      apiUrl="https://api.ourguide.com"
      productId="YOUR_PRODUCT_ID"
    />
  );
}
```

```vue Vue
<template>
  <OurguideWidget
    api-url="https://api.ourguide.com"
    product-id="YOUR_PRODUCT_ID"
  />
</template>

<script setup>
import { OurguideWidget } from "@ourguide/vue";
</script>
```

</CodeGroup>

<Warning>
  Replace `YOUR_PRODUCT_ID` with the actual Product ID from your dashboard.
</Warning>

## 3. Configure your agent

In the dashboard, set your agent name and instructions. Test responses in the live preview.

## 4. Add knowledge (optional)

In the [Knowledge Base](/knowledge-base), upload PDFs, crawl your website, or connect Notion.

## 5. Connect your API (optional)

In [API Integration](/api-integration), upload your OpenAPI spec, configure auth, and select endpoints.
