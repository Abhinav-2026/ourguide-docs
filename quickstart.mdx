## 1. Get your Product ID

Sign up at the [Ourguide Dashboard](https://dashboard.ourguide.com) and create a new product. Copy your **Product ID** from the settings page.

## 2. Add the widget to your site

Choose your preferred integration method:

<CodeGroup>

```html Script Tag
<script
  src="https://api.ourguide.com/static/ourguide-widget.js"
  data-api-url="https://api.ourguide.com"
  data-product-id="YOUR_PRODUCT_ID"
></script>
```

```jsx React
import { OurguideWidget } from "@ourguide/react";

function App() {
  return (
    <OurguideWidget
      apiUrl="https://api.ourguide.com"
      productId="YOUR_PRODUCT_ID"
    />
  );
}
```

</CodeGroup>

<Warning>
  Replace `YOUR_PRODUCT_ID` with the actual Product ID from your dashboard.
</Warning>

## 3. Add knowledge

Go to the **Knowledge Base** section in your dashboard and add sources:

- **Website** — Enter your site URL to crawl and index
- **Files** — Upload PDFs, docs, or text files
- **Notion** — Connect your Notion workspace

The agent will learn from these sources and use them to answer questions.

## 4. Test the widget

Open your site in a browser. You should see the Ourguide chat widget in the bottom-right corner. Try asking it a question about your product.

## 5. Monitor conversations

Go to the **Activity** page in your dashboard to see live conversations, review agent responses, and improve your knowledge base.
