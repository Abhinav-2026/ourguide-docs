## Welcome

The Ourguide API allows you to programmatically interact with the Ourguide platform.

## Base URL

```
https://api.ourguide.com/v1
```

## Authentication

All API requests require authentication using an API key. Include your key in the request headers:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.ourguide.com/v1/endpoint
```
